===
```BASH
bash <(curl -sL https://raw.githubusercontent.com/DavideSOGG/SSR_SS-libev_sh/main/tiziblog_ss_libev.sh)
```

bash <(curl -sL https://raw.githubusercontent.com/DavideSOGG/SSR_SS-libev_sh/main/ToyoDAdoubi_doubi_ssr.sh)
